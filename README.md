# Proyecto RAIA

Este proyecto consta de dos partes principales: una API backend (FastAPI) y una interfaz frontend (Streamlit).

### 1. Iniciar el Backend (API)

En la primera terminal, ejecuta:

python -m uvicorn api:app --reload

### 2. Iniciar el Frontend (Streamlit)

En la segunda terminal, ejecuta:

python -m streamlit run LA_streamlit.py

_Esto abrirá la aplicación en tu navegador web._
